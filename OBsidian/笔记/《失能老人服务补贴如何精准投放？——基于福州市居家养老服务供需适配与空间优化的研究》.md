# 《失能老人服务补贴如何精准投放？——基于福州市居家养老服务供需适配与空间优化的研究》

**以*长护险试点和养老服务消费补贴*为政策背景，利用公开追踪调查数据识别老年照护需求，结合福州市养老服务 POI 测度供需错配，并通过因果推断和政策仿真回答“财政补贴如何精准释放银发服务消费**

**照护需求识别 → 政策效应评估 → 服务供需错配 → 补贴资源优化**

谁需要→政策是否释放消费→服务供给不足→补贴去向

## 二、初步数据处理

### 1.CHARLS/CFPS 构建老年照护需求数据库

通过 XGBoost、随机森林和 SHAP 识别高照护需求老人，形成“老年照护需求风险指数”。

### 2.整理长护险试点政策数据

通过 DID、PSM-DID 和事件研究法评估长护险对老年家庭消费结构、服务消费和照护负担的影响

### 3.以福州市为案例做空间供需分析

爬取养老服务 POI，结合街道老年人口和公共服务资源，用 2SFCA、核密度分析和空间自相关识别服务供需错配区。

### 4.进行补贴政策仿真

比较平均补贴、重点人群补贴和精准补贴三种方案，测算财政支出、覆盖率、消费释放和供需改善效果。

### 1、老年照护需求识别

目标：判断哪些老人最可能需要长期照护服务。

方法：**随机森林 / XGBoost / LightGBM + SHAP解释**

解释变量：

年龄、健康状况、慢性病、ADL/IADL、是否独居、收入、医保、子女支持等。

解释变量：

是否存在失能；
是否有长期照护需求；
是否高医疗支出；
是否需要他人照料。

输出结果：

老年照护需求风险指数
高风险老人画像（聚类）
影响照护需求的关键因素排序


**需求供给错位**
### 2、长护险政策效应评估

目标：判断长护险是否释放了老年家庭的服务消费，是否减轻了照护负担。

方法：**DID 双重差分 + PSM-DID + 事件研究法 + 稳健性检验**

被解释变量：

家庭总消费；
医疗支出；
护理相关支出；
服务型消费；
子女经济支持；
家庭照护负担；
预防性储蓄倾向。

核心解释变量：

是否属于长护险试点地区
政策实施前后
处理组 × 政策后

输出结果：

长护险是否显著影响老年家庭消费结构
政策对哪些群体更有效
政策是否释放了银发服务消费

### 3、福州市养老服务供需适配分析

目标：判断福州养老服务资源是否与老年照护需求匹配。

方法：**2SFCA 两步移动搜索法 + 核密度分析 + Moran’s I 空间自相关 + 热点分析**

养老服务供给指数；
老年照护需求指数；
供需适配指数；
供需错配指数。

输出结果：

福州市养老服务供需适配地图；
高需求—低供给街道；
低需求—高供给街道；
优先建设或补贴区域。

空间公平问题可以作为市场供需错配的物理根源

### 4、**补贴政策仿真与资源配置优化**

目标：回答政府有限财政资金应该怎么投。

方法：**情景模拟 + 成本效果分析 + 多目标优化**

设计三种政策方案：

方案一：平均补贴。 所有符合条件老人同等补贴。

方案二：重点人群补贴。优先补贴高龄、失能、独居、低收入老人。

方案三：精准补贴。按照“照护需求风险 + 服务供给缺口 + 消费释放潜力”综合排序，优先补贴高需求低供给地区。

评价指标：

服务覆盖率；
财政支出；
单位财政支出带来的消费释放；
高风险老人覆盖比例；
供需错配指数下降幅度。
