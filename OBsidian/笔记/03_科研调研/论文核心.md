# 论文核心（更新版）

## 1. 研究主线

本文聚焦于 **weakly paired / misaligned H&E-to-IHC virtual staining**。  
与传统依赖强像素配准的 image-to-image translation 不同，我们将该任务建模为一个**条件化的连续时间恢复过程**：给定 H&E 图像作为条件输入，从噪声域出发，通过 **conditional rectified flow** 学习到 IHC 图像的生成轨迹。

整篇工作的主线只保留三部分：

1. **Conditional Restore Flow**  
    用 conditional rectified flow 取代传统 GAN 式虚拟染色主骨干，将 H&E→IHC 视为一个连续时间的条件恢复过程。
2. **Structure Constraint**  
    从 H&E 中提取具有病理意义的结构 proxy，作为 source-side morphology prior，在网络深层与 bottleneck 中进行多尺度结构锚定，并在 loss 端做 source structure consistency。
3. **Semantic Injection**  
    在 target-side 引入渐进式语义监督，不要求像素级对齐，而强调 pathology semantics 的稳定保持，用于缓解 consecutive slices 带来的 spatial inconsistency。

本工作**不把 OT 作为主方法组成部分**。OT 可以保留在 related work 或未来扩展里，但当前主方法已经足够闭环，不需要额外引入 OT 才能成立。这个取舍也和你当前模版中的判断一致。

---

## 2. 方法定位

### 2.1 问题定义

给定弱配对的 H&E–IHC 图像对 (xhe, xihc)(x_{he}, x_{ihc})(xhe​, xihc​)，其中两者通常来自 consecutive sections，因此存在天然的空间不一致与局部错位。本文的目标不是学习一个依赖严格像素监督的翻译器，而是学习一个在 **结构保持** 与 **病理语义一致性** 之间平衡的条件生成模型。

### 2.2 本文的核心观点

我们认为 weak pairing / misalignment 不是边角噪声，而是 virtual IHC 的核心难点。因此方法设计不应围绕“如何更强地拟合像素”，而应围绕：

- 如何让生成轨迹始终贴近 H&E 的原始组织结构；
- 如何在空间不完全对应的情况下仍然保住 IHC 的语义表达；
- 如何在不依赖 GAN 判别器的前提下稳定完成虚拟染色。

这也与你的现有模版判断一致：ASP 和 MICCAI 2024 的 weakly paired 相关工作证明了“弱配对是主问题”；PSPStain 则进一步说明“即使 spatial inconsistency 存在，也要保住 pathology semantics”。

---

## 3. 主要方法论

## 3.1 Conditional Rectified Flow 主骨干

我们采用 **conditional rectified flow (cRF)** 作为生成主骨干，将 H&E→IHC 建模为一个连续时间条件恢复问题。训练时，对目标 IHC 图像与噪声进行时间插值，网络学习预测速度场；推理时，从随机噪声出发，通过有限步 ODE 迭代恢复出目标 IHC。

相较于传统 GAN 骨干，这种建模方式的优势在于：

- 不依赖判别器进行对抗训练；
- 更适合与结构先验、时间条件、语义监督进行统一融合；
- 能把“生成过程的稳定性”直接纳入方法设计，而不仅是后验修补。

在叙事上，这一点也与模版里对 PASB 的判断一致：生成骨干创新本身就是一条可成立的路线，而你的切入点是 **cRF + 结构/语义解耦约束**。

---

## 3.2 Source-side Structure Constraint

为了避免 weakly paired 条件下生成结果偏离 H&E 的原始空间几何，我们从 source H&E 中提取一组具有病理物理意义的结构 proxy，作为 morphology prior：

- tissue mask
- hematoxylin-like map
- nuclei-like map
- edge map
- void mask

这些 proxy 不是用于替代图像输入，而是作为一条独立的结构条件分支，进入 `StructurePriorEncoder`，并在多尺度上与主干 U-Net 对齐。随后，结构特征通过 **gated residual injection + block modulation** 注入到 encoder 深层、bottleneck 与 decoder 关键层，形成对生成轨迹的结构锚定。

此外，在 loss 端，我们对生成 IHC 再次提取 proxy，并与 source HE 的结构 proxy 进行一致性约束，从而形成 source-side structure consistency。

这个设计的重点不是要求 H&E 与 IHC 逐像素一致，而是要求“生成图在组织轮廓、核分布、边缘与空洞结构上，仍然服从 source H&E 的 morphology layout”。这也与模版里对 UNIStainNet 和 topology-aware 路线的判断一致：你更接近的是 **source-side structural conditioning**，而不是 target-side pixel regression。

---

## 3.3 Target-side Semantic Injection

仅有 source structure 还不足以完成高质量 virtual IHC，因为 H&E 与 IHC 的关系不仅是几何映射，还包含 marker-specific pathology semantics。为此，我们在 target 侧引入渐进式语义监督：

- 通过语义分支或 proxy-like 目标统计来刻画 IHC 的病理表达特征；
- 采用 curriculum 方式逐步抬升语义约束权重；
- 避免一开始就用强语义损失压制主干生成过程。

这种设计的核心思想是：  
**不要求 target-side 强空间对齐，而强调 non-spatial / weakly aligned pathology semantics。**

这正是你的模版里最明确希望从 PSPStain 借用的一条线，即 **target-side non-spatial semantics**。同时，CCPL 提供了另一个可借鉴方向：病理通道统计一致性与跨通道相关性更适合作为增强项，而不是替代主干创新。

## 4. 技术栈

本文方法的技术栈可以概括为三层：

### 4.1 生成骨干层

- Conditional Rectified Flow
- Conditional U-Net backbone
- Continuous-time velocity prediction
- ODE-based iterative sampling

### 4.2 结构建模层

- Handcrafted pathology structure proxies
- StructurePriorEncoder
- Multi-scale structure feature pyramid
- Gated residual injection
- FiLM-like / scale-shift block modulation
- Source structure consistency loss

### 4.3 语义约束层

- Progressive semantic curriculum
- Pathology-aware semantic supervision
- Weakly aligned target-side semantics
- DAB / nuclei / histogram-related statistics

- PyTorch

## 5. 具体实现

## 5.1 输入与主干

输入由两部分构成：

- `noisy_target`：当前时间步下的噪声化目标状态
- `condition`：RGB H&E 图像

两者在通道维拼接后进入 `ConditionalUNet` 主干。模型采用多尺度 encoder–decoder 结构，在时间嵌入的调制下完成速度预测。

## 5.2 结构先验提取

对 source H&E 图像进行 proxy 提取，生成 5 通道结构条件图。该结构图不直接与主输入拼接，而是走独立的 `StructurePriorEncoder`，形成与主干各尺度对齐的结构特征：

- high-resolution morphology anchors
- mid-level tissue layout
- deep structural semantics
- bottleneck-level global morphology prior

## 5.3 多尺度结构注入

结构注入采用两级策略：

### (1) Gated residual injection

在 encoder 深层、bottleneck、decoder 关键层，将结构特征通过 1×1 projection 对齐到当前层通道空间，再使用可学习 gate 进行残差式融合：

x←x+αl⋅ϕ(sl) x \leftarrow x + \alpha_l \cdot \phi (s_l) x←x+αl​⋅ϕ(sl​)

其中 αl\alpha_lαl​ 为可学习门控系数，ϕ(⋅)\phi (\cdot)ϕ(⋅) 为 1×1 卷积投影。

### (2) Block modulation

在关键 block 内，再使用轻量的 scale-shift modulation，对 feature map 做条件调制：

x^=x⋅(1+γ(sl))+β(sl)\hat{x} = x \cdot (1 + \gamma (s_l)) + \beta (s_l) x^=x⋅(1+γ(sl​))+β(sl​)

其中 γ,β\gamma, \betaγ,β 由结构特征映射得到。

这种设计的目的，是让 structure prior 不仅“加进来”，而且“调进去”，从而使结构条件真正参与生成轨迹塑形。

## 5.4 语义课程学习

语义监督不从第 1 个 epoch 就强行加入，而是采用 curriculum 策略：

- 前期：以 flow 主任务与结构保持为主；
- 中期：逐步提升语义注入权重；
- 后期：在不破坏整体重建稳定性的前提下，强化 marker-related semantics。

这种安排是为了避免训练后期出现语义损失过强导致的模式崩塌。
